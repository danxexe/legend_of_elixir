## Install Scenic dependencies

```
brew update
brew install glfw3 glew pkg-config
mix archive.install hex scenic_new
```

## Create a new project

```
mix scenic.new legend_of_elixir
```

## Run

```
cd legend_of_elixir
mix do deps.get, scenic.run
```
